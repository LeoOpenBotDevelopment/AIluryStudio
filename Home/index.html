Projects

1: AIlury Studio

AIlury Programming Studio is a code editor for frontend developers. With AIlury Programming Studio, we are aiming to make coding HTML websites more accessible for mobile users by creating an intuitive UI, with a focus on tools to make coding quicker and maximize efficiency.

open a file in AIlury Programming Studio through your website by integrating this into your JavaScript code:

localStorage.setItem("AIluryStudioPendingFileName", "FileName.js");

localStorage.setItem("AIluryStudioPendingFileContent", "console.log('The File Content')");

localStorage.setItem("AIluryStudioPendingFileRun", "false"); //or true

localStorage.setItem("AIluryStudioPendingFileType", "js"); //html, css or js

//open AIlury Programming Studio

window.location.href = 'https://ailury.com';
